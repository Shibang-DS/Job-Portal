# Online Job Portal
Built a job portal where people can search for jobs as per their requirements according to the postings.
Recruiters can add any job postings in the Firebase database itself.

# Tech Stack Used
  1. Vite
  2. React.js
  3. Tailwind CSS
  4. Firebase
